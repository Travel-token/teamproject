# 여행 정산 앱 — React Native (Expo) 변환본


## 실행 방법
```bash
npm install
npx expo start
```

## 폴더 구조
```
App.tsx
src/
  theme/        # 라이트·다크 컬러 토큰, ThemeContext
  types.ts       # 공용 타입
  data/          # 목업 데이터
  navigation/    # 탭/스택 네비게이터
  components/    # Avatar, Fab, BottomSheetModal, RoomTabBar 등 공용 UI
  screens/       # 화면별 컴포넌트
```
